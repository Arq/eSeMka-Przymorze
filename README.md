## Description of the contents of the project folders
Fully digitally controlled railway station based on the [EX-CommandStation (Ready-to-Run or Do-It-Yourself) — DCC-EX](https://dcc-ex.com/ex-commandstation/index.html#) project with control of accessories (semitators and switches) using the DCC protocol and a digital desktop in the form of the JMRI application and a modular hardware desktop controlled by the I2C protocol
### Commandstation-EX-Configuration

Configuration files (myAutomation, myDevice ... ) for Commandstation-Ex

### JMRI

After downloading installing the [JMRI application](https://www.jmri.org/download/index.shtml) the deskt1op project files for the JMRI application should be copied to the \Users\<user-name>\JMRI folder.

### ICBlok

Elements used from the ICBlock project - a modular digital control panel project

### DaBus-MS16d
Occupancy detector module files
